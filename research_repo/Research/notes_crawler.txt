this command will create a text file in all directories, do not use this without severe caution!

. -type d -exec sh -c 'echo "Notes for this influence:" > "{}/notes.txt"' \;




this one is untested but creates only in leaf directories:

find . -type d ! -exec find {} -mindepth 1 -type d -print -quit \; \
-exec sh -c 'echo "Notes for this influence:" > "{}/notes.txt"' \;


"find . -type d" → look at all directories

"! -exec find {} -mindepth 1 -type d -print -quit \;"
 → exclude any directory that contains another directory

Remaining directories are leaf nodes only

"touch {}/notes.txt" → create notes.txt there if it doesn’t exist

Parent folders are skipped
Only edge directories get notes
Safe to re-run anytime
